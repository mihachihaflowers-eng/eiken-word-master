# 英単語学習アプリ 要件定義書

---

## 1. プロジェクト概要

| 項目 | 内容 |
|------|------|
| アプリ名 | 英検単語マスター（仮） |
| 目的 | 英検2級・準1級レベルの英単語・連語を効率的に暗記する |
| 動作環境 | Windows PC（Python製デスクトップアプリ） |
| GUIフレームワーク | **CustomTkinter**（モダンUI） |
| データ保存 | ローカルファイル（SQLite） |
| 単語データ | `eiken_vocabulary.md` を起動時に読み込んでSQLiteへインポートする |

---

## 2. 機能要件

### 2.1 レベル・バッチ管理機能

| 要件ID | 内容 |
|--------|------|
| LV-01 | レベルは「英検2級」「英検準1級」の2レベルから選択できる |
| LV-02 | 現在のレベルとバッチ番号をホーム画面に常時表示する |
| LV-03 | 単語・連語を合わせて100個ずつのセット（バッチ）で管理する |
| LV-04 | 現バッチの100個を全て「習得済み」にすると「次のバッチへ」ボタンが活性化される |
| LV-05 | 「次のバッチへ」ボタンを押すと、次のバッチが開始される |
| LV-06 | 取得する単語は過去に出題されていないものから100個選ぶ |
| LV-07 | 英検2級：最大4バッチ（計400語）、英検準1級：最大4バッチ（計400語）を用意する |
| LV-08 | 全バッチを習得した場合はその旨を通知する |

---

### 2.2 クイズ（一問一答）機能

| 要件ID | 内容 |
|--------|------|
| QZ-01 | 1セッションは10問とする |
| QZ-02 | 出題対象は現バッチの「未習得」単語・連語からランダムに選ぶ |
| QZ-03 | 各問題に1つの英単語（または連語）と6つの日本語訳の選択肢を表示する |
| QZ-04 | 正解は1つ、残り5つはランダムに選ばれた他の単語・連語の意味とする |
| QZ-05 | 選択肢を選んだ瞬間（クリック時）に即時正誤判定を行う |
| QZ-06 | **正解時**：ボタンを緑色で強調し、その単語・連語の意味を1〜3個表示する |
| QZ-07 | **不正解時**：選んだボタンを赤色で強調し、正解ボタンを緑色で示す |
| QZ-08 | 出題方向は英語 → 日本語のみとする |
| QZ-09 | 10問終了後、セッション結果（正答数・正答率）を表示してホーム画面へ戻る |
| QZ-10 | セッション途中でアプリを閉じた場合、次回起動時に中断したセッションを再開できる |
| QZ-11 | 各問題に「わからない」ボタンを表示する。押すと不正解扱い（連続正解リセット・不正解回数+1）となり、正解を緑色でハイライト表示する |

---

### 2.3 単語・連語 進捗管理機能

| 要件ID | 内容 |
|--------|------|
| WD-01 | 単語・連語ごとに「正解回数」「不正解回数」「連続正解回数」を記録する |
| WD-02 | 1度でも不正解になると「記憶できていないリスト」に追加する |
| WD-03 | 「連続正解2回」を達成した単語・連語は「習得済み」として登録し、以降は出題しない |
| WD-04 | 不正解になった場合、連続正解カウントは0にリセットする |
| WD-05 | 習得済みになった単語・連語は「記憶できていないリスト」から自動的に除外する |

---

### 2.4 単語一覧ページ

| 要件ID | 内容 |
|--------|------|
| VW-01 | 「習得済み単語一覧」ページを提供する（単語・意味・習得日・間違えた回数を表示） |
| VW-02 | 「記憶できていない単語一覧」ページを提供する（単語・意味・間違えた回数・連続正解数を表示） |
| VW-03 | 各一覧は間違えた回数・連続正解数などでソート・フィルタリングできる |
| VW-04 | 習得済み単語一覧の各行に「苦手に移す」ボタンを表示する。押すと習得解除・苦手単語リストへ移動する（逆方向の手動移動は不可） |
| VW-05 | 習得済み・苦手単語一覧の各行に「詳細」ボタンを表示し、単語詳細ポップアップを開ける |

---

### 2.5 暗記確認モード（集中特訓）

| 要件ID | 内容 |
|--------|------|
| DR-01 | 指定した「間違えた回数のしきい値」以上の単語・連語のみを出題する特訓モードを提供する |
| DR-02 | しきい値はユーザーが数値入力で指定できる（例：3回以上間違えた単語） |
| DR-03 | 出題ルールはクイズ機能（QZ章）と同様とする |
| DR-04 | セッション途中の中断・再開はクイズ機能と同様に対応する |

---

### 2.6 単語詳細表示機能

| 要件ID | 内容 |
|--------|------|
| DT-01 | 回答後（正解・不正解・わからない問わず）に「さらに詳しく表示」ボタンを表示する |
| DT-02 | ボタンを押すと詳細ポップアップを開き、発音記号・日本語翻訳・品詞別の意味・英語例文（最大2件）を表示する |
| DT-03 | 詳細情報は Google翻訳（品詞別の意味）と Free Dictionary API（発音記号・例文）から取得する |
| DT-04 | 取得した詳細情報は SQLite の `word_details` テーブルにキャッシュし、次回以降はネットアクセスしない |
| DT-05 | キャッシュ済みの単語はオフライン環境でも詳細表示できる |
| DT-06 | ~~詳細ポップアップ内に「この詳細情報を削除」ボタンを設ける~~ → 削除済み（全単語プリフェッチ済みのため不要） |
| DT-07 | 全単語の詳細情報を一括プリフェッチするスクリプト（`prefetch_details.py`）を提供する |

## 3. 単語データ仕様

| 項目 | 内容 |
|------|------|
| データソースファイル | `eiken_vocabulary.md`（同一ディレクトリに配置） |
| 読み込みタイミング | アプリ初回起動時（またはDBが空の場合）に `eiken_vocabulary.md` をパースしてSQLiteへインポートする |
| 収録内容 | 英単語および連語（コロケーション）を混在させる |
| データ構造（1件） | 英単語/連語、品詞、日本語訳（最大3個）、レベル（2級 or 準1級）、バッチ番号 |
| 英検2級 | 100個 × 4バッチ = 計400件（`eiken_vocabulary.md` の2級セクションより取得） |
| 英検準1級 | 100個 × 4バッチ = 計400件（`eiken_vocabulary.md` の準1級セクションより取得） |
| バッチ割り当て | ファイル内の出現順に100件ずつバッチ1〜4へ自動割り当てする |
| 重複排除 | 同一単語・連語が複数バッチに登場しないよう管理する |

---

## 4. 非機能要件

| 要件ID | 内容 |
|--------|------|
| NF-01 | GUIフレームワーク：**CustomTkinter** を使用する |
| NF-02 | データはローカルに永続保存する（**SQLite**） |
| NF-03 | Python 3.10以上で動作すること |
| NF-04 | セッション中断データはSQLiteに保存し、次回起動時に復元する |
| NF-05 | `eiken_vocabulary.md` が同一ディレクトリに存在しない場合はエラーメッセージを表示して終了する |
| NF-06 | DB初期化済みの場合は `eiken_vocabulary.md` の再読み込みをスキップする（既存進捗を保持） |
| NF-07 | PyInstaller で単体 exe としてビルドできる。Python 未インストール環境でも動作する |
| NF-08 | 単語詳細のキャッシュは `eiken.db` に保存する。DB ごと別 PC に持ち運ぶことでオフライン詳細表示も継続できる |

## 5. 画面構成

```
ホーム画面
├── レベル表示（英検2級 / 英検準1級）
├── 現在のバッチ番号表示
├── 習得進捗バー（XX / 100）
├── クイズ開始ボタン（未完了セッションがある場合は「セッションを再開」）
├── 暗記確認モードボタン（しきい値入力付き）
├── 習得済み単語一覧ボタン
├── 記憶できていない単語一覧ボタン
└── 「次のバッチへ」ボタン（100個習得時のみ活性）

クイズ画面
├── 問題番号（X / 10）
├── 英単語・連語の表示
├── 選択肢ボタン × 6
├── わからないボタン（押すと不正解扱い・正解をハイライト表示）
├── 正誤フィードバック表示
├── さらに詳しく表示ボタン（回答後に表示 → 詳細ポップアップを開く）
└── 中断ボタン

習得済み単語一覧画面
├── 単語リスト（単語・意味・習得日・間違えた回数）
├── 各行：「苦手に移す」ボタン・「詳細」ボタン
└── ソート・フィルタ機能

記憶できていない単語一覧画面
├── 単語リスト（単語・意味・間違えた回数・連続正解数）
├── 各行：「詳細」ボタン
└── ソート・フィルタ機能

単語詳細ポップアップ
├── 発音記号
├── 日本語翻訳
├── 品詞別の意味（日本語）
├── 英語例文（最大2件）
└── 「閉じる」ボタン
```

## 6. 確定済み事項

| # | 確認事項 | 決定内容 |
|---|----------|----------|
| ① | 単語データのソース | `eiken_vocabulary.md` を起動時にパースしてSQLiteへインポートする |
| ② | 単語数 | 英検2級：100個×4バッチ（計400語）、英検準1級：100個×4バッチ（計400語）。合計800語（単語・連語混在） |
| ③ | GUIフレームワーク | **CustomTkinter**（モダンUI） |
| ④ | 記憶できていないリストの定義 | 1回でも不正解 → 追加。習得済みになったら自動除外 |
| ⑤ | 連語の出題形式 | 単語と同じ形式（選択肢6択） |
| ⑥ | レベル | 英検2級・英検準1級の2レベルから選択可能 |
| ⑦ | セッション中断 | 中断可能。次回起動時に再開できる |
| ⑧ | 出題方向 | 英語 → 日本語のみ |
| ⑨ | 単語詳細情報の取得元 | Google翻訳（品詞別の意味）+ Free Dictionary API（発音記号・例文） |
| ⑩ | 単語詳細のキャッシュ | SQLite `word_details` テーブルに保存。一括プリフェッチスクリプトで事前取得可能 |
| ⑪ | スタンドアロン配布 | PyInstaller で単体 exe 化。`eiken.db`・`eiken_vocabulary.md` と同フォルダに置いて使用 |
